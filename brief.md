## 1. ЦЕЛЬ И АУДИТОРИЯ

### Цель сайта

Привлекать B2B-клиентов и конвертировать их в заявки через Telegram. Лендинг решает три задачи одновременно:
1. **Объяснить** — кто это и что делает (за 3 секунды)
2. **Доказать** — что умеет (через кейсы с метриками)
3. **Вызвать действие** — написать в Telegram (через минимальное трение)

### Целевая аудитория

**Основная:** владельцы малого и среднего бизнеса (10–100 сотрудников), которые хотят:
- Новый сайт / лендинг / интернет-магазин
- Веб-приложение или MVP стартапа
- Автоматизацию через AI-агентов (поддержка, лиды, обработка заявок)

**Вторичная:** стартапы на ранних стадиях, технические директора, менеджеры проектов.

**Психографика клиента:**
- Занят, у него мало времени — решение принимает быстро или откладывает навсегда
- Боится нарваться на фрилансера, который пропадёт с предоплатой
- Не понимает AI — думает «это дорого и не для меня»
- Хочет видеть конкретные цифры, не абстрактные «качественно и в срок»
- Принимает решение через Telegram, а не форму

**Что НЕ является аудиторией:** другие разработчики, рекрутеры крупных корпораций, клиенты с бюджетом менее $300.

*Моноширинный шрифт. Мигающий курсор `|` в конце — typewriter-эффект.*

Три кнопки-якоря под подзаголовком:
```
[→ Нужен сайт]    [→ Нужно приложение]    [→ Нужен AI-агент]
```
*Каждая кнопка плавно скроллит к соответствующей карточке в секции Services.*

#### Мини-социальное доказательство
```
60+ проектов    5 лет опыта    Клиенты из 8 стран
```
*Три цифры в ряд, крупный шрифт, разделены точками или `·`.*

- **Основная:** `Обсудить проект →` — акцентный цвет, ведёт в Telegram
- **Вторичная:** `Смотреть работы` — ghost-кнопка, якорь на `#portfolio`

#### Визуальный элемент (Hero Background)
Анимированный dot-grid фон с subtle glow-пятном, следующим за курсором:
```css
background-image: radial-gradient(circle, rgba(124,58,237,0.25) 1px, transparent 1px);
background-size: 28px 28px;
/* + spotlight: radial-gradient следует за --cx, --cy через mousemove */
```

**Альтернатива:** GLSL-шейдер с noise-градиентом (если позволяют ресурсы разработки).

- Видео-превью запускается без звука

- Появляются кнопки «Смотреть кейс» и «Live»

**3D Tilt-эффект (опционально):**
```js
card.addEventListener('mousemove', e => {
  const { left, top, width, height } = card.getBoundingClientRect();
  const x = (e.clientX - left) / width - 0.5;
  const y = (e.clientY - top) / height - 0.5;
  card.style.transform = `perspective(800px) rotateY(${x*12}deg) rotateX(${-y*12}deg) translateY(-8px)`;
});
```

**Макет:** Asymmetric masonry grid. На мобиле — одна колонка.

**Список вопросов:**

  Лендинг: 1–2 недели. Веб-приложение: 3–8 недель.

  Это цифровой сотрудник: отвечает клиентам, квалифицирует заявки,

### Анимации

**Обязательные:**

| Элемент | Анимация | Библиотека |
|---------|----------|-----------|
| Hero headline | Typewriter или word-by-word reveal | CSS / JS |
| Секции при скролле | Fade + slide up (stagger) | Framer Motion / CSS |
| Карточки услуг | Hover: подъём + glow | CSS |
| Dot-grid фон | Spotlight следует за курсором | Vanilla JS |
| Цифры в About | Count-up при появлении | Framer Motion |
| Плавный скролл | Momentum scroll | Lenis |
| Активный пункт меню | Highlight при скролле | Intersection Observer |

**Scroll-driven (нативный CSS):**
```css
@keyframes fadeSlideUp {
  from { opacity: 0; transform: translateY(40px); }
  to   { opacity: 1; transform: translateY(0); }
}
.animate-on-scroll {
  animation: fadeSlideUp linear both;
  animation-timeline: view();
  animation-range: entry 0% entry 40%;
}
```

**Spotlight cursor (dot-grid секции):**
```js
document.addEventListener('mousemove', e => {
  document.documentElement.style.setProperty('--cx', `${e.clientX}px`);
  document.documentElement.style.setProperty('--cy', `${e.clientY}px`);
});
```

**Spring-анимация кнопок:**
```css
transition: transform 0.4s cubic-bezier(0.34, 1.56, 0.64, 1);
```

**Обязательно отключать при prefers-reduced-motion:**
```css
@media (prefers-reduced-motion: reduce) {
  *, *::before, *::after { animation-duration: 0.01ms !important; transition-duration: 0.01ms !important; }
}
```
